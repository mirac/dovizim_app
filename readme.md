# "Dovizim" Android Uygulamasi

Bu uygulama Flex.io API'lari ve Retrofit2 kütüphanesi kullanilarak Miraç Satiç tarafindan gelistirilmistir. 

Her iki saniyede bir döviz kurlari ekranda güncellenir, tüm islemler multi-thread sekilde arka planda yapilir, sistemde herhangi bir kilitlenme olmaz.

**Önemli not:** Flex.io'nun ücretsiz sagladigi API hizmet paketinde 1000 istek\aylik limit vardir. Eger veriler düzgün görüntülenmezse lütfen CurrencyActivity.java dosyasinin 61. satirindaki API erisim anahtarini degistiriniz.

### Tesekkürler
Android Studio
Flex.IO
Retrofit2
